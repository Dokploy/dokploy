Dokploy is a free, self-hosted PaaS (Platform as a Service) designed to simplify the deployment and management of applications and databases. This repository is based on the official Dokploy with added multi-language interface (i18n) support.

## ‚ú® Features

Dokploy provides a comprehensive set of capabilities to help you manage applications and databases more easily on your own servers:

- **Application Deployment**: Supports multiple languages and runtimes including Node.js, PHP, Python, Go, Ruby, and more.
- **Database Management**: Built-in creation and management of databases including MySQL, PostgreSQL, MongoDB, MariaDB, Redis, and more.
- **Backups**: Configure automatic backups to external storage for databases.
- **Docker Compose**: Native support for Docker Compose, perfect for complex multi-service applications.
- **Multi-node Clustering**: Cluster management based on Docker Swarm, enabling multi-node scaling.
- **Template Marketplace**: One-click deployment of open-source applications (Plausible, Pocketbase, Cal.com, etc.).
- **Traefik Integration**: Automatic routing and load balancing, integrated with Traefik.
- **Real-time Monitoring**: Monitor CPU, memory, disk, network, and other resource usage.
- **Docker Management**: Easily deploy and manage containers.
- **CLI / API**: Manage resources through command line or API.
- **Multi-server Management**: Unified management of deployments on remote servers.
- **Self-hosted**: Fully self-hosted on your VPS or physical machine.

## üöÄ Quick Start

Run the following command on a fresh Linux server to install Dokploy:

```bash
curl -sSL https://raw.githubusercontent.com/Frankieli123/dokploy-i18n/main/install.sh | bash
```

The script above will automatically pull the images:

- Main panel service: `a3180623/dokploy-i18n:<version>`

To use the official Dokploy Cloud, visit: [Dokploy Cloud](https://app.dokploy.com).  
For more documentation, refer to the official docs: [docs.dokploy.com](https://docs.dokploy.com).
