### [Tuple, aplikasi berbagi layar terkemuka untuk pengembang](https://tuple.app/dokploy)
[Tersedia untuk MacOS dan Windows](https://tuple.app/dokploy)<br>

Dokploy adalah PaaS (Platform as a Service) gratis dan self-hosted yang dirancang untuk menyederhanakan penyebaran dan pengelolaan aplikasi dan database. Repositori ini didasarkan pada Dokploy resmi dengan dukungan antarmuka multi-bahasa (i18n) yang ditambahkan.

## âœ¨ Fitur

Dokploy menyediakan serangkaian kemampuan komprehensif untuk membantu Anda mengelola aplikasi dan database dengan lebih mudah di server Anda sendiri:

- **Penyebaran Aplikasi**: Mendukung berbagai bahasa dan runtime termasuk Node.js, PHP, Python, Go, Ruby, dan lainnya.
- **Pengelolaan Database**: Pembuatan dan pengelolaan database bawaan termasuk MySQL, PostgreSQL, MongoDB, MariaDB, Redis, dan lainnya.
- **Backup**: Konfigurasi backup otomatis ke penyimpanan eksternal untuk database.
- **Docker Compose**: Dukungan native untuk Docker Compose, sempurna untuk aplikasi multi-layanan yang kompleks.
- **Clustering Multi-node**: Pengelolaan cluster berbasis Docker Swarm, memungkinkan penskalaan multi-node.
- **Pasar Template**: Penyebaran satu klik aplikasi open source (Plausible, Pocketbase, Cal.com, dll.).
- **Integrasi Traefik**: Routing dan load balancing otomatis, terintegrasi dengan Traefik.
- **Pemantauan Real-time**: Pantau penggunaan CPU, memori, disk, jaringan, dan sumber daya lainnya.
- **Pengelolaan Docker**: Mudah menyebarkan dan mengelola container.
- **CLI / API**: Kelola sumber daya melalui baris perintah atau API.
- **Pengelolaan Multi-server**: Pengelolaan terpadu penyebaran di server jarak jauh.
- **Self-hosted**: Sepenuhnya self-hosted di VPS atau mesin fisik Anda.
