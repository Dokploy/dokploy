Dokploy biedt een uitgebreide reeks mogelijkheden om u te helpen applicaties en databases eenvoudiger te beheren op uw eigen servers:

- **Applicatie-implementatie**: Ondersteunt meerdere talen en runtime-omgevingen, waaronder Node.js, PHP, Python, Go, Ruby en meer.
- **Databasebeheer**: Ingebouwde aanmaak en beheer van databases, waaronder MySQL, PostgreSQL, MongoDB, MariaDB, Redis en meer.
- **Backups**: Configureer automatische backups naar externe opslag voor databases.
- **Docker Compose**: Native ondersteuning voor Docker Compose, perfect voor complexe multi-service applicaties.
- **Multi-node clustering**: Clusterbeheer gebaseerd op Docker Swarm, maakt multi-node schaalvergroting mogelijk.

- **CLI / API**: Beheer resources via de opdrachtregel of API.
