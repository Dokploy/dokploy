## 介绍

本指南详细介绍了如何为您的应用程序设置域名。

## 要求

如果您没有域名，请首先参阅 `Web域名设置` 部分。

## 创建 DNS 记录

1. 将 `A` 记录添加到您的 DNS 设置中：
  - **名称:** 输入您要指向的路线 (例如, `app` 到 `app.yourdomain.com`).
  - **值:** 输入服务器的 IP 地址， 例如 `1.2.3.4`.

## 应用程序域名设置

1. 导航至应用程序的仪表板。
2. 转到 “域名” 选项卡并单击 “添加域名”。
3. 输入您在上一步中创建的 DNS 记录 （例如:`app.yourdomain.com`）。
