The Certificates section alows you to manage SSL/TLS certificates efficiently.

## Actions

- **Create**: Create a new certificate.
- **Delete**: Delete an existing certificate.

### Create a Certificate

To create a new certificate, fill out the form with the following details:

- **Name**: Enter a name for the certificate (this can be anything you choose).
- **Certificate Data**: Provide the certificate details.
- **Private Key**: Enter the private key.

<Callout>
This action will create the certificate on the server. For more detailed configuration and management, use the `Traefik` tab.
</Callout>
